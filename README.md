# 🩺 Glycosense – Diabetes Management System

## 📌 Overview

**Glycosense** is an AI-powered diabetes management system designed to help users monitor and manage their blood glucose levels efficiently.
It provides tools for logging meals, tracking glucose readings, and generating personalized dietary recommendations.

---

## 🚀 Features

* 📊 Track blood glucose levels
* 🍽️ Log meals and nutrition data
* 🤖 AI-based dietary recommendations
* 📱 Multi-module structure (frontend, backend, mobile)
* 🔒 Secure environment configuration

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/sahil676r-bot/Glycosense.git
cd Glycosense
```

---

### 2️⃣ Create Virtual Environment (Recommended)

```bash
python -m venv venv
venv\Scripts\activate   # Windows
source venv/bin/activate  # Mac/Linux
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Setup Environment Variables

* Copy `.env.example` → `.env`
* Add your configuration values (API keys, DB, etc.)

---

### 5️⃣ Run the Project

```bash
python main.py
```
